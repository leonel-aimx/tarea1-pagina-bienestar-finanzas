# 💻 Simulador de Perfil Financiero Bienestar

🚀 Proyecto web tipo mockup enfocado en educación financiera para personas en México que reciben apoyos del Bienestar.

---

## 📌 Descripción

Este proyecto consiste en una aplicación web estática que simula una herramienta de análisis financiero básico, diseñada para ser accesible, clara y fácil de usar.

Permite al usuario ingresar datos personales y financieros para generar un perfil financiero simulado acompañado de recomendaciones prácticas.

Está enfocado en usuarios sin conocimientos técnicos, incluyendo adultos mayores.

---

## 🎯 Objetivo del Proyecto

Desarrollar una interfaz intuitiva que:

- Fomente la educación financiera básica  
- Sea accesible para todo tipo de usuarios  
- Simule una herramienta digital con enfoque social  
- Aplique buenas prácticas de desarrollo web  

---

## ⚙️ Funcionalidades

### 📥 Captura de datos
- Edad  
- Tipo de apoyo (beca, pensión, programa bienestar)  
- Monto mensual  
- Hábitos financieros (ahorro, deudas, uso de apps)  

### 📊 Resultado simulado
- Perfil financiero:
  - 🔴 Riesgo  
  - 🟡 Transición  
  - 🟢 Crecimiento  
- Explicación sencilla  
- Plan de acción en 3 pasos  

---

## 🧠 Lógica del sistema (simulada)

La clasificación del perfil se basa en condiciones simples:

- 🔴 Riesgo: No ahorra y tiene deudas  
- 🟡 Transición: Ahorra pero tiene deudas  
- 🟢 Crecimiento: Ahorra y no tiene deudas  

> ⚠️ Nota: No se utilizan bases de datos ni APIs. Toda la lógica es local.

El diseño está enfocado en:

## 🚫 Limitaciones

- No cuenta con backend  
- No guarda información  
- No realiza análisis real  
- No está conectado a servicios externos
